# Simple Grade Book Project
This is a program that allows user to manage a gradebook of students, as well as keep information on specific students.

## How To Use
- Just run from main.py

## Project Features
- Manage gradebook and students
- Keep grades for certain students
- Calculate student grade average
- Calculate student letter grades

## Planned Project Features
- Calculate student GPA
- Calculate class average
- Find highest and lowest grades in class
